## [126邮箱深色模式](./126-mail-darkmode/126-mail-darkmode.css)

> 针对网易126邮箱(mail.126.com)的深色模式的样式表，使用`prefers-color-scheme`，跟随浏览器主题进行深色/浅色模式自动切换。

### 功能

* 为网易126邮箱添加深色模式样式，使用`prefers-color-scheme`，当浏览器切换为深色模式时，自动生效。
